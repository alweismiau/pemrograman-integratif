
# Implementasi gRPC dan Protobuf dengan js dan firebase

## Apa yang digunakan 
- js
- Firebase
- Protobuf

## Cara menjalankan code
    1. Run server.js (mengaktifkan server dan gRPC)
    2. Run client.js di terminal berbeda 
    3. Buka firebase (firestore) dan masukan input CRUD pada client.js
